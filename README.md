## ⑤工夫した点・こだわった点
- これまでやってきたことをもう一度やることを意識。Firebase＋localstrage、ランダム関数などを活用、今回は２回目だったので理解は深まったという気がする
- 今回はHTML、JS、CSSの役割をそれぞれ区分、JSは定義まで、見た目はCSSでと言う後から修正しやすい形にこだわった。
- コメントの表示を自分と他の人で左右区分するところなので相当苦労した。JS側でlocalstrageを参照してクラスを付与、そのクラスに応じてCSSで位置を振り分けるという分担に。

## ⑥難しかった点・次回トライしたいこと（又は機能）
- ただただCSSがむずかった。JSでFirestrage使うところまではなんとかなったが、単純に表示をJSでやってしまうとその後のCSS側での処理が複雑になりすぎるorJS側が長文化するリスクがあったので、役割を分ける方法を考えるのが一番苦労した。
